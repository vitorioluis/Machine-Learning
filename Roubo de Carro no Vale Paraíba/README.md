
# Projeto de Data Science de roubo de veículos no Vale Paraíba

#### Objetivo:
Obter uma visão geral de onde mais acontece os roubos, quais são as ruas, bairros, cidades com maiores incidência de roubos, 
observar evolução ou de redução disso, verificar quantos carros são encontrados, enfim ter uma visão geral disso no Vale Paraíba.  

#### Projeto está em fase de obtenção dos dados que estão disponíveis em http://www.ssp.sp.gov.br/transparenciassp/Consulta.aspx
